Leafstack Learning Roadmap

If you’re a fresher and everything feels confusing, this roadmap gives you one clean path: learn basics → build Django apps → explain them like an interview story.

Bonus: it’s designed for small-batch + buddy learning (so you don’t ghost after 3 days).

Buddy Mode (student-to-student)

Learning alone is hard. So we keep it simple:

  • Pick a buddy (friend or batchmate)
  • Daily 30 mins “ship something small” session
  • Weekly demo: show what you built in 2 minutes
Invite a Friend to Join You

Tip: even if you’re both beginners, you’ll progress faster together.

Phase 1: Python Foundations

Goal: feel confident writing basic logic.

  • Programming basics (variables, loops, functions)
  • Python syntax + logic with small practice
  • How to think: break problems into steps
Student prompt to ask your buddy

“Explain today’s topic in 30 seconds. I’ll do the same.”

Phase 2: Django & Web

Goal: understand how real websites are built.

  • How the web works (request/response)
  • Django structure (apps, URLs, views, templates)
  • Database basics + CRUD pages
Mini-checkpoint (simple but powerful)

If you can build one CRUD feature (add/edit/delete + list page), you are already ahead of most freshers.

Phase 3: Real Projects

Goal: build portfolio projects + learn to explain them.

  • Solo project (you build end-to-end)
  • Group project (teamwork + roles + Git workflow basics)
  • Interview explanation skills (story format)
How students help each other here

Buddy reviews: you demo your feature; your buddy asks questions like “why did you do it this way?” This makes you interview-ready.

Common Doubts (quick answers)

I’m from non-CS. Is this okay?

Yes. We start from zero and focus on practice + projects. Non-CS students usually do great when they follow a steady plan.

Do I need a strong laptop?

A basic laptop is enough for Python + Django. We keep tooling simple.

What if I miss days?

You won’t be kicked out. We keep a clear “next step” so you can restart without shame.
